La data para cargar debe tener la siguiente estructura:

  • cedula de identidad (solo numer, sin puntos)
  • primer nombre
  • primer apellido
  • segundo nombre
  • segundo apellido
  • codigo de carrera
  • codigo de modalidad de ingreso


Cada dato deben estar encerrado entre comillas dobles ( " " ) separado entre comas ( , ) y al final de la linea un salto normal de linea (un simple enter) ejemplo:

"2222222","maria", "rodriguez", "valentina", "perez", "613", "2"
"5555555","pedro", "pereira", "alexander", "colmenares", "613", "2"

Este formato de datos se consigue exportado la hoja de excel sin ningun tipo de cabezera de datos a formato csv,
y en las columnas donde los datos son numericos, se deben selecinar y en propiedades de celda colocar tipo de datos de texto
